## 📌 Project Overview
The Random Password Generator is a Python-based application that generates strong and secure passwords randomly. This project allows users to create secure passwords by specifying the desired password length.

## 🎯 Objectives
- Generate strong and secure passwords.
- Enhance user password security.
- Allow customization of password length.
- Demonstrate Python programming concepts.

## 🛠️ Technologies Used
- **Python**
- Random Module
- String Module
- Functions

## ✨ Features
✔ Generate random passwords instantly  
✔ User-defined password length  
✔ Includes uppercase letters (A-Z)  
✔ Includes lowercase letters (a-z)  
✔ Includes numbers (0-9)  
✔ Includes special characters (@, #, $, %, etc.)  
✔ Easy-to-use command-line interface

## 🚀 How to Run the Project
1. Install Python on your system.
2. Download or clone the project files.
3. Open the project folder in VS Code or any Python IDE.
4. Run the following command:

```bash
python password_generator.py
```

## 📖 Working
1. The user enters the desired password length.
2. The program randomly selects characters from letters, digits, and symbols.
3. The selected characters are combined to create a secure password.
4. The generated password is displayed to the user.
